# scripts

Each subfolder here is an **author collection** — a set of PowerShell scripts maintained by one contributor.

## Convention

- Name your folder after your **GitHub handle** (lowercase), e.g. `scripts/octocat/`.
- Put your `.ps1` files directly inside it (top level; PaletteShell does not recurse into subfolders).
- Include a `README.md` in your folder describing who you are and listing your scripts.
- You own everything inside your folder. Don't edit other authors' folders except to fix a genuine bug (open a PR and explain).

## Adding your collection

1. Create `scripts/<your-handle>/`.
2. Copy in `scripts/_template/README.md` and fill it out.
3. Drop your `.ps1` scripts in.
4. Open a pull request.

See [CONTRIBUTING.md](../CONTRIBUTING.md) for script formatting details.
