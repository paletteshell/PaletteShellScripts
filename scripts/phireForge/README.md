## Scripts

| Script | What it does |
|---|---|
| `View-DiskSpace.ps1` | Show disk space statistics for all fixed drives, rendered as Markdown. |
| `Generate-Password.ps1` | Generate a random password from a form of options (length, character sets, exclude ambiguous) and copy it to the clipboard. |
| `List-Processes.ps1` | Browse/search running processes by memory use; select one to copy its PID. |
| `Stop-Process.ps1` | Terminate a process by PID or name, with a confirmation prompt. Pairs with `List-Processes.ps1`. |
| `View-EnvironmentVariables.ps1` | Dump the current environment variables as a fenced Markdown code block. |
| `View-SystemStatistics.ps1` | Show operating system, hardware, memory, disk, network, and process statistics rendered as Markdown. |
| `View-EventLogErrors.ps1` | Show recent Windows Event Log errors, rendered as Markdown. |
| `View-CertificateExpiry.ps1` | Show local certificates that are expired or expiring soon, rendered as Markdown. |
| `View-TimeZones.ps1` | Show current time across selected time zones, rendered as Markdown. |
| `Flush-DNS.ps1` | Clear the Windows DNS resolver cache. |
| `CheatSheet-RegularExpressions.ps1` | Show a practical regex reference with tokens, flags, recipes, PowerShell examples, and debugging tips. |
| `CheatSheet-Markdown.ps1` | Show a polished Markdown reference with syntax examples, rendered output, and practical tips. |
| `Clipboard-PreviewMarkdown.ps1` | Take Markdown text from the clipboard and display it rendered in Markdown mode. |
| `Clipboard-LinesToCsv.ps1` | Take multi-line clipboard text and join it into a single comma-delimited line. |
| `Clipboard-LinesToSingleQuoteCsv.ps1` | Take multi-line clipboard text, single-quote each line, and join comma-delimited (SQL `IN`-ready). |
| `Clipboard-LinesToDoubleQuotedCsv.ps1` | Take multi-line clipboard text, double-quote each line, and join comma-delimited (CSV-style). |
