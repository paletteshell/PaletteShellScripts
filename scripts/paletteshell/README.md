# PaletteShell (@paletteshell)

Official scripts maintained by the PaletteShell team. This folder is a good place to look for well-documented examples of the script format.

- GitHub: [@paletteshell](https://github.com/paletteshell)

## Scripts

| Script | What it does |
|---|---|
| `Base64-Decode.ps1` | Decode Base64 clipboard text and copy back. |
| `Base64-Encode.ps1` | Encode clipboard text to Base64 and copy back. |
| `Clear-TempFiles.ps1` | Delete old files in your user TEMP folder, then report the count and space freed. |
| `Clipboard-RemoveDuplicateLines.ps1` | Remove duplicate lines from clipboard text. |
| `Clipboard-SortLines.ps1` | Sort clipboard lines alphabetically. |
| `Clipboard-ToCSV.ps1` | Convert multiline clipboard text to comma-delimited, quote-wrapped CSV. |
| `Clipboard-ToLowerCase.ps1` | Convert clipboard text to lowercase. |
| `Clipboard-ToUpperCase.ps1` | Convert clipboard text to uppercase. |
| `Clipboard-TrimLines.ps1` | Remove leading and trailing whitespace from each line. |
| `Clipboard-UnixTimestamp.ps1` | Generate the current Unix timestamp and copy it to the clipboard. |
| `Clipboard-UrlDecode.ps1` | URL decode clipboard text and copy back. |
| `Clipboard-UrlEncode.ps1` | URL encode clipboard text and copy back. |
| `Export-ProcessList.ps1` | Snapshot running processes as CSV and open the output as a file. |
| `Generate-GUID.ps1` | Generate a new GUID and show it as a copyable result. |
| `Get-PublicIP.ps1` | Look up this machine's public IP address and show it as a copyable result. |
| `Git-Branches.ps1` | List local Git branches as a searchable, pickable list. |
| `Json-Format.ps1` | Pretty-print JSON from the clipboard and copy back. |
| `Json-Minify.ps1` | Compress JSON from the clipboard and copy back. |
| `Restart-Explorer.ps1` | Restart the Windows Explorer shell after confirmation. |
| `System-Report.ps1` | Show a quick system summary rendered as Markdown. |
| `Text-Transform.ps1` | Apply customizable transformations to clipboard text. |
